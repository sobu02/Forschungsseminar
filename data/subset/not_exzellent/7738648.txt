AUO2010 = :Das Ergebnis, das der Spieler bei den Australian Open 2010 erzielt hat. Dabei werden folgende Abkürzungen verwendet: S = Sieg, F = Finale, HF = Halbfinale, VF = Viertelfinale, AF = Achtelfinale, 3R / 2R / 1R = 3. / 2. / 1. Runde, 3Q / 2Q / 1Q = 3. / 2. / 1. Qualifikationsrunde. Falls der Spieler nicht an dem Turnier teilgenommen hat, so kann dieser Parameter leer- oder weggelassen werden. Falls das Turnier noch aussteht, so wird hier eingetragen. Falls der aktuelle Stand während eines Turniers angezeigt werden soll, so sollte die momentane Runde eingeklammert werden (z.B.
